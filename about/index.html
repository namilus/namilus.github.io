IBM Watson Dispatch Manager

The system we were asked to develop, as part of a group was a call screening agent that uses the IBM Watson technology. The agent connects callers to an Emergency Medical Dispatcher (EMD) if they are reporting an incident that is not yet known about or if they have further information about one. This agent will solve the problem of emergency services becoming overwhelmed with several incoming calls all reporting the same incident.

For the project, I was in charge of the development of the database, that was developed using MongoDB. An intermediate RESTful API was developed in order to query the database via HTTP requests. I was also in charge of developing the Watson Assistant and it’s Cloud Function using IBM’s Watson Natural Language Processing facilities.

The supervisor from IBM, had the following to say about the quality of the project: “You have been very enthusiastic about tackling the challenge and I think you have made much more progress than I would have expected from a student project.”

– Brian O Donovan, IBM Software Engineer brian_odonovan@ie.ibm.com
